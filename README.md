Amazon Clone (HTML & CSS)
Project Overview

This is a front-end clone of Amazon built using HTML and CSS. It showcases a responsive layout with a navigation bar, search bar, and product cards, focusing on clean UI and design.

Features

Responsive navigation bar

Product listing with images, names, and prices

Search bar (UI only)

Clean, modern design using HTML & CSS

Technologies Used

HTML5

CSS3 (Flexbox & Grid for layout)

How to Use

Clone or download this repository.

Open index.html in your browser.

Explore the responsive layout and product listings.
